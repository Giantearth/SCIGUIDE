# SCI 转移与移乘：各节段核心技术

转移能力是实现患者功能独立的核心指标。训练重点随神经损伤平面（NLI）的下降，从“完全依赖”过渡到“代偿转移”，最终实现“独立移乘”。

## 1. 核心节段的功能区别

| 损伤节段 | 移乘能力预期 | 核心技术/工具 |
| :--- | :--- | :--- |
| **C1 - C4** | **完全依赖** | 需 2 人以上介助或使用机械升降机。 |
| **C5** | **重度依赖** | 需他人协助摆放体位，利用万能袖带辅助支撑。 |
| **C6** | **功能分水岭** | **核心：[[肌腱固定式抓握 (Tenodesis grasp)]]** 与惯性。部分年轻、体能好者可在滑板辅助下实现介助/独立转移。 |
| **C7 - T1** | **基本独立** | **核心：肱三头肌（伸肘）**。具备伸肘推起能力，能独立完成床-轮椅转移。 |
| **T2 - L2** | **完全独立** | 具备良好的躯干平衡。可进行复杂地形转移（如轮椅到地面、轮椅到浴缸）。 |
| **L3 及以下** | **坐站转化潜力** | **核心：股四头肌（伸膝）**。具备从坐位直接站起并步行的潜力。 |

## 2. 床-椅转移 (Bed-to-Chair) 训练核心

### 技术要点
- **利用滑板 (Sliding Board)**：针对 C6 及不完全性损伤患者，滑板可减少摩擦力，是早期过渡到独立转移的必备工具。
- **高度补偿**：训练初期，应保持床面略高于轮椅面（或反之），利用重力辅助移动。
- **搬腿先行**：指导患者（或介助者）先将一侧下肢搬至目标面上，以此带动骨盆转动。
- **头-平移原则**：利用“头向相反方向摆动”的生物力学原理，带动臀部向目标方向移动。

### 肩关节保护 (关键)
> [!CAUTION] 治疗师注意
> 频繁转移极易导致肩袖损伤。
> - 避免在不平等的界面间强行移乘。
> - 严禁通过拽拉头顶上方的固定物来提拉身体。
> - 尽可能减少每日不必要的转移次数。

## 3. 坐-站转移 (Sit-to-Stand) 训练核心

### 适应证与前提
- **循环耐受**：必须先通过斜面床训练，确认无严重的 [[直立性低血压 (Orthostatic Hypotension)]]。
- **肌力要求**：
    - **L3 关键点**：伸膝肌力 $\ge 3$ 级是独立坐站转化的先决条件。
    - 若肌力不足，需佩戴 KAFO（长下肢支具）固定膝关节。

### 动作核心
- **重心前移**：指导患者躯干充分前倾，使肩部超过膝关节垂线（鼻尖过脚尖）。
- **支撑点分布**：利用助行器或平行杠提供前方支撑。
- **辅助站立**：对于高位截瘫，治疗师需从前方锁定患者膝关节，防止跪倒。

## 4. 转移过程中的红旗征
- **脊柱稳定性**：[[创伤性脊髓损伤 (TSCI)]] 术后未获准许前，严禁进行涉及脊柱大角度扭转的转移。
- **皮肤剪切力**：移乘时严禁拖拽身体，必须推起或滑动，防止诱发 [[压力性损伤 (压疮)|骶尾部压疮]]。
- **自主神经反应**：T6 以上损伤在移乘中若出现剧烈头痛，应立即停止，排查 [[自主神经反射异常 (AD)]]。
