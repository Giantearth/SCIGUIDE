# SCI 康复目标总览

康复目标的设定需基于神经损伤平面及残损分级，并结合原发病因的特殊性进行动态调整。

## 1. 按 [[ASIA 残损分级 (AIS) 简介|AIS 等级]] 链接目标
- **[[ASIA 残损分级 (AIS) 简介#1. 核心定级标准|完全性损伤 (AIS A)]] 康复目标**：侧重于利用残存功能的“代偿”训练及并发症预防。
- **[[ASIA 残损分级 (AIS) 简介#1. 核心定级标准|不完全性损伤 (AIS B-D)]] 康复目标**：侧重于残存神经通路的“诱导”与功能强化训练。

## 2. 按 [[简介与病因分类|疾病病因]] 修正目标
- **[[创伤性脊髓损伤 (TSCI)]]**：目标达成前提是脊柱生物力学稳定。
- **[[非创伤性脊髓损伤 (NTSCI)#缺血性脊髓损伤 (Ischemic SCI)|缺血性脊髓损伤]]**：康复强度需严格参考心血管耐受力（如主动脉手术后）。
- **[[非创伤性脊髓损伤 (NTSCI)#肿瘤性脊髓损伤 (Neoplastic SCI)|肿瘤性脊髓损伤]]**：目标设定需考虑病理性骨折风险，多为维持性康复。

## 3. 专业分工任务矩阵 (点击跳转)
- [[物理治疗 (PT) 目标]]：大粗大运动、移动与行走。
- [[作业治疗 (OT) 目标]]：精细功能、ADL 与环境适应。
- [[言语治疗 (ST) 目标]]：吞咽安全与交流。
- [[呼吸康复目标]]：气道管理与通气独立。
