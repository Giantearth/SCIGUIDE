# 颈段脊髓损伤 (Cervical SCI)

颈段损伤导致**四肢瘫 (Tetraplegia)**。根据残存的关键肌和康复目标，临床上将其精细化细分为以下子节段：

## 1. 高位颈髓 (C1 - C4)：呼吸支持区
- **C1 - C3**：
    - **状态**：完全依赖机械通气或膈肌起搏器。
    - **康复潜力**：通过嘴、眼球或语音驱动辅助技术。
- **C4**：
    - **状态**：**膈肌功能保留**，通常可脱离呼吸机；生活完全依赖。
    - **康复潜力**：利用下巴、嘴或头部驱动电动轮椅。

## 2. 中低位颈髓 (C5 - C6)：手功能代偿区
- **C5**：关键肌为**肱二头肌 (屈肘)**。
    - **状态**：肩部和屈肘有动力；但无法伸肘，无手功能。
    - **康复潜力**：利用自助具实现“手到嘴”活动（如自主进食）。
- **C6**：关键肌为**桡侧伸腕长肌 (伸腕)**。
    - **核心标志**：**[[肌腱固定式抓握 (Tenodesis grasp)]]**。
    - **康复潜力**：通过伸腕联动手指被动抓握，有望实现独立翻身、转移和最小辅助下的自我照料。

## 3. 低位颈髓 (C7 - T1)：轮椅独立区
- **C7**：关键肌为**肱三头肌 (伸肘)**。
    - **状态**：具备伸肘推起能力，对转移至关重要。
    - **康复潜力**：手动轮椅独立，大部分 ADL 自理。
- **C8 - T1**：关键肌为**指深屈肌及小指展肌**。
    - **状态**：手指抓放功能恢复。
    - **康复潜力**：无需辅助的自我照料，能操作标准交流工具。
