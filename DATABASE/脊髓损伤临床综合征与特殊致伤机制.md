# 脊髓损伤临床综合征与特殊机制

在临床实践中，部分患者的表现并不符合单纯的节段受损规律，而是呈现出特定的综合征表现。这些情况对康复潜力的预后判断具有重要意义。

## 1. 核心临床综合征 (Clinical Syndromes)

### [[脊髓半切综合征 (Brown-Séquard Syndrome)]]
- **病理机制**：脊髓横断面的侧半部受损。
- **常见病因**：多见于**刀刺伤**等穿透性损伤。
- **临床特征**：
    - 同侧：损伤平面以下运动功能丧失（肢体瘫痪）、深感觉（位置觉、振动觉）消失。
    - 对侧：损伤平面以下痛觉和温觉消失。
- **康复潜力**：此类损伤通常属于**运动不完全性损伤**，恢复潜力相对较好。指南建议一旦血流动力学稳定，应尽早启动移动能力训练。

### 中央型脊髓综合征 (Central Cord Syndrome)
- **病理机制**：脊髓中央管周围受损。
- **常见病因**：老年人原有颈椎管狭窄，在遭受轻微外伤（如跌倒导致颈椎过伸）时诱发。
- **临床特征**：
    - **上肢重于下肢**：典型的“手部瘫痪明显、双腿尚能行走”的表现。
    - 伴有远端肌肉受累明显和不同程度的感觉丧失。
- **康复手段**：重点在于**精细作业治疗 (OT)** 以恢复手功能，同时进行步态训练。

## 2. 罕见/特殊致伤机制

### [[无影像学异常的脊髓损伤 (SCIWORA)]]
- **特征**：常见于儿童和青少年。X线或CT检查结果阴性，但临床存在明确的神经受损体征。
- **康复注意**：尽管影像学正常，但仍需按脊髓损伤规范管理，严格制动并动态观察神经体征变化。

### [[电击伤与雷击伤性脊髓损伤]]
- **特征**：由高压电或雷击引起。
- **特殊红旗**：神经症状可能具有**延迟性**和**进行性**。起初可能仅表现为截瘫，数小时或数天后可能恶化为四肢瘫。
- **管理建议**：急性期需延长临床观察期，不能因早期症状轻微而掉以轻心。

### [[潜水减压病性脊髓损伤]]
- **特征**：见于水肺潜水员。神经症状通常在上升后数分钟出现，或在随后的飞行（气压变化）中诱发。
- **急救康复联动**：怀疑此病时，除常规管理外，必须**紧急转运至高压氧舱单位**进行减压治疗，这是挽救神经功能的黄金手段。

## 3. 心理与功能鉴别

### [[癔症性瘫痪 (Hysterical Paralysis)]]
- **定义**：属于转化障碍，表现为瘫痪但无器质性损伤依据。
- **鉴别评估：SIC 测试**
    - 方法：患者仰卧，治疗师将其双膝被动屈曲，双脚平放于床面。
    - 阳性表现：若撤除支撑后，患者双膝能自主维持并拢状态而不倒向两侧，高度怀疑为癔症性。
- **康复手段**：
    - **行为疗法**：以鼓励为主，轻柔引导患者恢复正常功能。
    - **多学科协作**：需心理医生及精神科介入，成功处理心理因素后，躯体功能常可迅速恢复。

## 4. 伴随性骨病风险
- **[[强直性脊柱炎 (AS)]] 与 DISH**：
    - 此类患者脊柱极脆。即便遭受**极低能量**的跌倒（如平地摔倒），也极易发生波及脊髓的骨折。
    - 治疗师在面对此类患者时，即便外伤看似微小，也必须先排除骨折红旗，严禁暴力牵伸。
