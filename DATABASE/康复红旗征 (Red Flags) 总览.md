# SCI 康复红旗征 (Red Flags)

## 3. 康复训练通用红旗 (Systemic Red Flags)

- [[特殊禁忌与药物红旗|异位骨化 (HO)]]：关节活动度突然受限伴局部红肿热。
- [[压力性损伤 (压疮)]]：骨突处不可压灭的红斑。
- [[直立性低血压 (Orthostatic Hypotension)]]：从仰卧位转变为直立位（坐位或站立）过程中或变换后 **3 分钟内**，血压出现显著下降
