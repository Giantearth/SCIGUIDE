# ASIA 残损分级 (AIS) 简介

脊髓损伤神经学分类国际标准（ISNCSCI）是目前全球通用的评估工具，其中的 ASIA 残损分级（AIS）用于界定损伤的严重程度。

## 1. 核心定级标准
- **A 级 (完全性损伤)**：最低骶节（S4-5）无任何感觉或运动功能保留。
- **B 级 (感觉不完全损伤)**：神经损伤平面以下（包括 S4-5）有感觉功能，但无运动功能。
- **C 级 (运动不不完全损伤)**：平面以下保留运动功能，且超过一半的关键肌肌力小于 3 级。
- **D 级 (运动不完全损伤)**：平面以下保留运动功能，且至少一半的关键肌肌力大于或等于 3 级。
- **E 级 (正常)**：感觉和运动功能均正常（仅适用于既往有损伤史的患者）。

## 2. 关键术语
- **神经损伤平面 (NLI)**：身体两侧感觉正常且关键肌肌力 $\ge 3$ 级的最低脊髓节段。
- **骶残留 (Sacral Sparing)**：S4-5 节段存在轻触觉、针刺觉或肛门括约肌自主收缩，是区分完全与不完全损伤的核心。
- **[[脊髓休克 (Spinal Shock)]]**：损伤后早期的反射消失状态。休克期结束（以球海绵体反射 BCR 恢复为标志）后的评估结果才具有预后判断价值。
