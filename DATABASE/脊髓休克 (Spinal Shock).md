# 脊髓休克 (Spinal Shock)

脊髓休克是指急性脊髓损伤后，损伤平面以下脊髓功能（包括反射和自主神经功能）的暂时性完全丧失状态。

## 1. 核心临床特征
- **反射消失**：损伤平面以下的腱反射、浅反射（如腹壁反射）以及骶段反射（如球海绵体反射、肛门反射）全部消失。
- **肌肉状态**：表现为典型的**弛缓性瘫痪**（软瘫），肌张力极低。
- **二便功能**：出现尿潴留（无尿意及排尿反射）和大便失禁/嵌塞。

## 2. 持续时间
- **变动范围大**：脊髓休克通常在伤后立即出现。
- **时间跨度**：可持续**数天至数月**不等（通常为数周）。
- **结束标志**：临床上以**球海绵体反射 (BCR)** 的恢复作为脊髓休克结束的首要标志。随后，损伤平面以下的原始反射会逐渐转变为亢进状态（硬瘫）。

## 3. 康复评估注意事项 (治疗师必读)

> [!IMPORTANT] 评估的可靠性
> 在脊髓休克期间进行的 ASIA (AIS) 评定**不能作为最终预后判断的依据**。
> - **假性完全损伤**：在休克期内，由于反射消失，不完全性损伤（AIS B-D）可能表现为类似完全性损伤（AIS A）的征象。
> - **重测要求**：指南建议治疗师应**每日检查 BCR 反射**。只有在 BCR 恢复、确认脊髓休克结束后，所做的神经学定级才具有客观的预后预测价值。

## 4. 关键鉴别诊断：脊髓休克 vs 神经源性休克

治疗师在急性期巡视时，必须区分这两种“休克”概念，因为其处理优先级完全不同：

| 特征 | 脊髓休克 (Spinal Shock) | 神经源性休克 (Neurogenic Shock) |
| :--- | :--- | :--- |
| **性质** | **神经反射系统**暂时抑制 | **循环/血流动力学**障碍 |
| **表现** | 反射消失、弛缓性瘫痪 | **低血压**、**心动过缓** |
| **常见平面** | 任何节段损伤均可出现 | 多见于 **T6 及以上** 损伤 |
| **对康复影响** | 影响神经评定的准确性 | 威胁生命，是早期动员的**红旗禁忌** |

## 5. 早期介入建议
- **良肢位摆放**：休克期肌肉完全无力，极易发生压疮和关节挛缩，需严格执行每 2 小时翻身。
- **膀胱管理**：此阶段通常需要留置尿管。一旦脱离休克期，应尽早评估间歇导尿（CIC）的可行性。
