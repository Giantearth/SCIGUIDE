# 步行与移动能力：节段、肌力与目标

脊髓损伤患者的步行潜力主要取决于**神经损伤平面 (NLI)**、**AIS 残损分级**以及**下肢关键肌的肌力**。

## 1. 步行可能性与 AIS 分级预后
- **AIS A (完全性)**：T2-L2 节段患者仅能在完全支撑或助行器辅助下进行极短距离步行；腰骶段以下大部分患者可步行。
- **AIS C (运动不完全性)**：50岁以下的 AIS C 级患者，绝大多数在出院时能恢复步行能力 [2]。
- **AIS D (运动不完全性)**：几乎所有 AIS D 级患者在出院时均具备步行能力（无论是否使用辅助具）。

## 2. 步行分级定义 (临床目标)
根据功能独立程度，步行目标分为以下三类：
1. **治疗性步行**：利用矫形器或助行器在平行杠内或室内短距离移动，主要目的在于维持骨密度和心理康复。
2. **功能性步行**：可在室内独立行走，但在室外长距离移动仍需依赖轮椅。
3. **社区水平步行**：能自主在户外、不平整路面行走，并能跨越障碍物或马路。

## 3. 详细节段康复目标与肌力要求

| 损伤节段         | 关键肌群 (肌力 $\ge 3$ 级) | 预期移动/步行目标                                           |
| :----------- | :------------------ | :-------------------------------------------------- |
| **C1 - C4**  | 仅保留颈部/膈肌功能          | 生活完全依赖；使用下巴、嘴或头部控制驱动**电动轮椅**。                       |
| **C5**       | 肱二头肌 (屈肘)           | 生活依赖；使用手指控制**电动轮椅**；平地无法划动手动轮椅。                     |
| **C6 - C8**  | 伸腕/伸肘/手指关键肌         | 可能独立翻身及转移；使用**手动轮椅**，但在长距离或复杂环境下常需电动轮椅。             |
| **T1 - T12** | 腹肌/躯干关键肌            | 手动轮椅独立；**T2-L2** 患者在完全支撑或助行器辅助下可进行**短距离步行**。        |
| **L2**       | 髂腰肌 (屈髋)            | 具备**功能性步行**潜力；通常需长下肢支具 (KAFO) 或腋拐辅助。                |
| **L3**       | 股四头肌 (伸膝)           | **关键分水岭**：伸膝肌力 $\ge 3$ 级是脱离长支具、改用短支具 (AFO) 行走的先决条件。 |
| **L4 - S1**  | 胫骨前肌/趾长伸肌/腓肠肌       | 大部分患者可实现**社区水平步行**；可能仅需 AFO 或无需支具。                  |

## 4. 步行训练方法与介入时机
治疗师应根据患者分期选择合适的训练手段：
- **减重步行训练 (BWSTT)**：利用减重平板进行早期步态诱导。
- **平行杠内步行**：包括摆至步、四点步和摆过步训练。
- **水中步行训练**：建议在伤后或术后 **4 周**进行，利用浮力减轻负荷。
- **康复机器人**：利用机器人辅助矫形器进行重复性步态训练。

## 5. 治疗师注意：步行红旗征
> [!CAUTION] 禁忌与风险
> - **下肢 DVT**：一旦确诊或高度怀疑深静脉血栓，严禁进行站立和步行训练。
> - **异位骨化 (HO)**：关节周围红肿热痛且活动度硬性受限时，应暂停高强度负重训练。
> - **稳定性未明**：创伤性 SCI 手术前或脊柱评分 $\ge 5$ 分（SLIC/TLICS）时，禁止起立训练。
> - **直立性低血压**：首次站立或步行训练时，若收缩压下降 $\ge 20$ mmHg，应立即复位平卧。
