# SCI 多学科康复治疗重点

本笔记详细阐述 PT、OT、ST 及呼吸康复的核心干预措施，并建立与 [[康复目标总览]] 的深度链接。

## 1. 物理治疗 (PT) 治疗重点
PT 侧重于大运动功能重建、移动能力及脊柱稳定性维护。
- **早期动员与制动平衡**：在确保脊柱稳定性（参照 [[创伤性脊髓损伤 (TSCI)]] 的稳定性评分）前提下，尽早启动床上翻身（原木翻身法）及关节活动度 (ROM) 训练。
- **肌力增强训练**：重点针对神经损伤平面以上的残存肌力及代偿肌（如背阔肌）进行抗阻练习。
- **转移与轮椅技巧**：涵盖床-轮椅间移乘、轮椅平衡以及跨越障碍物（如路边石）的训练。
- **步行诱导**：对 [[ASIA 残损分级 (AIS) 简介#1. 核心定级标准|不完全性损伤 (AIS B-D)]] 患者利用减重平板、水中步行或机器人辅助进行步态再学习。

## 2. 作业治疗 (OT) 治疗重点
OT 侧重于上肢功能优化、日常生活自理 (ADL) 及环境改造。
- **ADL 自理训练**：根据不同节段潜力训练进食、饮水、修饰（洗脸刷牙）及更衣。
- **辅助具配置与应用**：针对 C5 损伤制作**万能袖带 (Universal Cuff)**，针对 C6 损伤诱导 **[[肌腱固定式抓握 (Tenodesis grasp)]]**。
- **环境改造咨询**：评估患者回归家庭后的门口坡道、卫生间及呼叫按钮（眼动或语音控制）的可及性。

## 3. 言语治疗 (ST) 治疗重点
ST 侧重于吞咽安全、呼吸辅助下的交流及伴随认知评估。
- **吞咽功能管理**：对颈髓损伤、佩戴 [[康复红旗征 (Red Flags) 总览|哈罗氏架]] 或气管切开者进行正式评估，防止误吸性肺炎。
- **交流功能重建**：为 [[颈段脊髓损伤 (C1-T1)|C1-C3]] 呼吸机依赖患者建立替代性交流系统 (AAC)，针对构音障碍进行训练。
- **伴随认知筛查**：识别 [[创伤性脊髓损伤 (TSCI)]] 合并脑外伤带来的认知、执行功能障碍。

## 4. 呼吸康复治疗重点
呼吸康复是 [[颈段脊髓损伤 (C1-T1)|高位损伤]] 及 [[胸段脊髓损伤 (T2-T12)|高胸段损伤]] 急性期的生命保障 。
- **气道清除技术**：评估咳嗽峰值流速 (CPF)。若 CPF < 160 L/min，需由治疗师协助进行 **手动助咳 (Quad coughing)** 或应用机械排痰 (MI-E)。
- **呼吸肌训练**：针对膈肌进行抗阻训练（吸气肌），并训练胸锁乳突肌、斜方肌以补偿胸式呼吸缺失 。
- **撤机管理**：与医师协作，在肺活量 > 800 mL 时执行循序渐进的呼吸机撤离计划。
